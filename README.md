## Featured Projects

### 🤖 PatchPilot — Autonomous Code-Fixing Agent for GitHub Issues
Autonomous agent that localizes bug-relevant files via hybrid retrieval and runs a test-driven self-correction loop. Benchmarked end-to-end on SWE-bench Verified (500 real issues) at a 23% resolution rate and $0.42 per resolved issue.

**Tech Stack:** Python · tree-sitter · Anthropic API · pytest · SWE-bench  
🔗 https://github.com/tabs929/PatchPilot

---

### 🔎 Recall — Production RAG System with Eval Harness
Hybrid retrieval pipeline (BM25 + BGE-large dense embeddings with cross-encoder reranking) over a 12K-document corpus, served with sub-400ms p95 latency. Includes a RAGAS-based eval harness and grounded citation generation.

**Tech Stack:** FastAPI · Qdrant · BM25 · BGE · RAGAS · Streamlit  
🔗 https://github.com/tabs929/Recall

---

### 🛡 Sentinel — Observable Multi-Turn Tool-Using Agent
Multi-turn agent orchestrating GitHub, Linear, and Slack via Anthropic tool-use, with retry logic across 11 failure modes, input/output guardrails, persistent session state, and full OpenTelemetry tracing.

**Tech Stack:** Python · Anthropic tool-use · OpenTelemetry · SQLite · tenacity  
🔗 https://github.com/tabs929/Sentinel

---

### 📄 ContractIQ — AI-Powered Contract Intelligence Platform
Full-stack contract intelligence platform with a RAG pipeline over a vector store, 30+ REST APIs, and Redis-backed asynchronous document ingestion.

**Tech Stack:** React (TypeScript) · FastAPI · LangChain · Qdrant · Redis · Docker  
🔗 https://github.com/tabs929/contractIQ

---

### 🎟 Event Discovery Platform
Cloud-native event engine on Google Cloud Run integrating Ticketmaster and Spotify, with Redis token caching and geospatial search via geohashing.

**Tech Stack:** React (TypeScript) · Node.js · Express · MongoDB · Google Cloud  
🔗 https://github.com/tabs929/EventfInder-react

---

### 🖼 VisSeqBench — Holistic Benchmarking for Multimodal LLMs
Automated evaluation framework benchmarking multimodal LLM logical reasoning and visual-sequence processing across 500+ inputs, with semantic evaluation metrics using synonym graph normalization.

**Tech Stack:** Python · HuggingFace Transformers  
🔗 https://github.com/tabs929/VisSeqBench
